Audrey Link Relations

"schema" rel

The "schema" relation is used to indicate a link to a JSON Schema document. Collections include such links to describe the schema for each type of child resource that can be added to the collection.

Every schema link object has a "name" property containing a string representating the name of an Object type.

While on the topic of creating new resources, note that collections have a state property "_factory" whose value is an object with the following properties:

Property Notes
schemas A list of the object types that can be created in this collection. The list contains strings that can be correlated with the names of the "schema" links.
method The HTTP method to be used to create new child objects. For naming collections this will be PUT. For non-naming collections it will be POST.
