Audrey Link Relations

"file" rel

The "file" relation is used to indicate a link to download a file (stored in the MongoDB GridFS).

Every file link object has a "name" property containing a string representation of the file's ObjectId. As such, the "name" can be used to uniquely identify a specific file link. The ObjectId strings can be cross-referenced with "FileId" values found in an Object's state properties to determine the corresponding file link for a given property.

Each file link object has a "type" property containing a string representing the MIME type of the file.
